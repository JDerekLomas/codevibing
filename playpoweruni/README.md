# PlayPower Universe 🚀

An educational game package featuring a Japanese gaming aesthetic with a 3D universe interface and typing-based learning challenges.

## Features

### 🌌 Interactive 3D Universe
- **Cosmic solar system** with glowing sun at the center
- **Planets as learning domains** - each subject is a unique celestial body
- **Hover scaling effects** - planets grow 1.8x when hovered
- **Click-based navigation** - no hover labels, click to open side panel
- **Orbital paths** and **ring systems** for visual variety
- **Floating animation** and **particle effects** for immersion
- **Dynamic star field** with thousands of stars
- **Nebula clouds** and cosmic phenomena
- **Space-themed lighting** with sun illumination

### 🎮 Cosmic Typing Challenge Games
**All planets feature typing-based challenges with:**
- **Timed challenges** with countdown timers
- **Real-time character feedback** (green for correct, red for mistakes)
- **WPM (Words Per Minute) tracking**
- **Accuracy monitoring**
- **Progressive difficulty** across challenges
- **Space-themed mission briefings** with Japanese/English mix

### 🪐 Planetary Learning Systems

1. **Numbers Planet** 🔢 (Blue Planet)
   - **Size:** Large (1.2 units)
   - **Features:** Solid blue sphere, no rings
   - **Curriculum:** Mathematical formulas, arithmetic concepts, terminology

2. **Geometry Planet** 📐 (Green Ringed Planet)
   - **Size:** Medium (1.0 units)
   - **Features:** Green planet with orbital ring system
   - **Curriculum:** Geometric theorems, shape properties, spatial relationships

3. **People Planet** 👥 (Orange Giant)
   - **Size:** Largest (1.3 units)
   - **Features:** Warm orange surface, no rings
   - **Curriculum:** Emotional intelligence, social psychology, communication

4. **AI Planet** 🤖 (Purple Ringed Planet)
   - **Size:** Medium (1.1 units)
   - **Features:** Purple planet with advanced ring system
   - **Curriculum:** Artificial intelligence, machine learning, technology

5. **History Planet** 📚 (Red Planet)
   - **Size:** Medium-Large (1.15 units)
   - **Features:** Cratered red surface, no rings
   - **Curriculum:** Historical events, timelines, civilizations

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Game Progression

- Start with basic domains (Numbers, People)
- Complete games to earn points
- Unlock new domains as you progress:
  - 50 points: Geometry
  - 100 points: History
  - 200 points: AI & Technology

## Controls

### Globe Navigation
- **Click and drag**: Rotate the globe
- **Scroll**: Zoom in/out
- **Click on shapes**: Select a learning domain

### Game Controls
Each mini-game has its own controls, but generally use:
- **Mouse/Touch**: Select answers
- **Enter key**: Submit answers (where applicable)
- **Back button**: Return to globe

## Technologies Used

- **Next.js 14**: React framework
- **TypeScript**: Type safety
- **React Three Fiber**: 3D graphics
- **Three.js**: 3D rendering engine
- **Tailwind CSS**: Styling
- **Web Audio API**: Sound effects
