# Monster Hunt

Monster Hunt is an educational game that teaches coordinate plane concepts while having fun capturing monster emojis.

## Educational Standards

This game aligns with the following math standards:
- 6.NS.C.5: Describe quantities with positive/negative numbers; interpret 0 in context
- 6.NS.C.6: Extend number lines & coordinate planes to rational numbers (opposites, reflections, plotting)
- 6.NS.C.7: Order & compare rational numbers; interpret absolute value
- 6.NS.C.8: Graph points in all quadrants; use absolute value for horizontal/vertical distance

## Game Modes

1. **Flashlight Mode**: Click to shine a light and find monsters at specific coordinates.
2. **Slider Mode**: Use X and Y sliders to align targeting lines at coordinates.

## Difficulty Levels

1. **1 Quadrant**: Only positive X and Y values (Quadrant 1).
2. **2 Quadrants**: Introduces negative X values (Quadrants 1 & 2).
3. **4 Quadrants**: Full coordinate plane with positive and negative values (All Quadrants).

## Running the Project

```bash
# Install dependencies
npm install

# Run the development server
npm run dev

# Build for production
npm run build

# Start production server
npm run start
```
